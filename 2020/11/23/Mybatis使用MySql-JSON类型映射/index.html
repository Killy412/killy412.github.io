MyBatis 中使用自定义转换器处理 json 类型

mysql 在 5.7.8 版本以后就支持了 json 这种数据类型,在程序中处理的方法有很多,可以直接用 string 去接收 json 类型,然后再转. 也可以使用自定义转换器,需要继承BaseTypeHandler<>类,然后在配置文件中加个扫描路径,在使用的地方指定转换器.

  1. 使用处指定类型转换器

MySql 常用 Json 函数

  • json 是数组的话,使用$[0]来确定元素
  • json 是对象的话,使用$.today来确定元素
